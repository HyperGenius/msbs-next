# 戦略AI高度化 - デモンストレーション実行結果

このドキュメントは、実装された戦略AI機能の実際の動作を示すデモンストレーションの結果です。

## シナリオ設定

同じ戦場状況で、異なる戦術設定がターゲット選択にどのように影響するかを検証します。

### 戦場状況

**プレイヤー機体**
- 名前: プレイヤー機
- 位置: (0, 0, 0)
- HP: 120/120
- 装備: ビームライフル（威力35、射程800m）

**敵機体**

1. **ザク（近・弱）**
   - 位置: 250m先
   - HP: 60/60（満タン）
   - 装備: ザクマシンガン（威力12）
   - 戦略価値: 72.0
   - 脅威度: 0.40

2. **ガンダム（遠・強）**
   - 位置: 650m先
   - HP: 180/180（満タン）
   - 装備: ビームライフル（威力55）
   - 戦略価値: 235.0
   - 脅威度: 0.71

3. **ジム（中・損傷）**
   - 位置: 400m先
   - HP: 35/100（ダメージ大）
   - 装備: ビームスプレーガン（威力28）
   - 戦略価値: 128.0
   - 脅威度: 0.58

## 各戦術での選択結果

### 1. CLOSEST（近接優先）

```
【選択されたターゲット】
  >>> ザク（近・弱） <<<

【選択理由】
  プレイヤー機がターゲット選択: ザク（近・弱） (戦術: CLOSEST, 距離: 250m)
```

**解説**: 最も近い敵（250m）を選択。従来のシンプルな近接優先戦術。

---

### 2. WEAKEST（弱敵狙い）

```
【選択されたターゲット】
  >>> ジム（中・損傷） <<<

【選択理由】
  プレイヤー機がターゲット選択: ジム（中・損傷） (戦術: WEAKEST, HP: 35)
```

**解説**: 現在HPが最も低い敵（35/100）を選択。確実に撃破できる敵を優先する戦術。

---

### 3. STRONGEST（強敵優先）

```
【選択されたターゲット】
  >>> ガンダム（遠・強） <<<

【選択理由】
  プレイヤー機がターゲット選択: ガンダム（遠・強） (戦術: STRONGEST, 戦略価値: 235.0)
```

**解説**: 戦略価値が最も高い敵（235.0）を選択。高HP・高火力の強敵を優先排除。

---

### 4. THREAT（脅威度優先）

```
【選択されたターゲット】
  >>> ガンダム（遠・強） <<<

【選択理由】
  プレイヤー機がターゲット選択: ガンダム（遠・強） (戦術: THREAT, 脅威度: 0.71)
```

**解説**: 脅威度が最も高い敵（0.71）を選択。自機への危険性を最優先。
遠いにも関わらず、高火力により高い脅威度と判定された。

## 考察

### 戦術の使い分け

1. **CLOSEST（近接優先）**
   - 素早く交戦開始したい場合
   - 移動時間を最小化したい場合
   - 単純明快な戦術

2. **WEAKEST（弱敵狙い）**
   - 確実に敵の数を減らしたい場合
   - 被弾リスクを減らしたい場合
   - 撃破による戦況改善を重視

3. **STRONGEST（強敵優先）**
   - 戦略的に重要な敵を先に排除
   - 長期的な脅威を早期に除去
   - 高価値ターゲットを優先

4. **THREAT（脅威度優先）**
   - 自機の生存を最優先
   - 自機への危険性が高い敵を排除
   - 防御的な立ち回りに最適

### 脅威度の評価

このシナリオでは、ガンダムが以下の理由で最高の脅威度（0.71）と評価されました：

- **高火力**: 武器威力55（他の敵の約2倍以上）
- **距離補正**: 650mと遠いが、火力の高さがそれを上回る
- **計算式**: `(55 / 120) * (1000 / 650) = 0.71`

一方、ザクは近い（250m）が火力が低い（12）ため、脅威度は0.40に留まりました：
- **計算式**: `(12 / 120) * (1000 / 250) = 0.40`

## まとめ

同じ戦場状況でも、戦術設定によって選択されるターゲットが劇的に変化することが確認できました。これにより、プレイヤーは状況に応じた適切な戦術を選択し、より戦略的な戦闘が可能になります。

単純な「最近接ターゲット」方式から、**状況判断に基づく高度な行動**が実現されました。
