## カラーパレット

### カウントダウンタイマー
- **背景**: グリーン〜ブルーのグラデーション (from-green-900/20 to-blue-900/20)
- **枠**: 明るいグリーン (border-green-500)
- **テキスト**: ライトグリーン (text-green-300)

### エントリーダッシュボード（未エントリー）
- **ENTRYボタン**: グリーン〜ブルーのグラデーション
- **参加者数パネル**: ブルー〜パープルのグラデーション
- **プログレスバー**: ブルー〜パープル

### エントリーダッシュボード（エントリー済み）
- **背景**: グリーン系 (bg-green-900/30)
- **枠**: 明るいグリーン (border-green-500)
- **ステータスインジケーター**: パルスする緑ドット
- **キャンセルボタン**: レッド系

### バトル結果モーダル（WIN）
- **背景**: グリーン〜ブルーのグラデーション
- **枠**: グリーン (border-green-500)
- **テキスト**: ライトグリーン (text-green-300)

### バトル結果モーダル（LOSE）
- **背景**: レッド〜グレーのグラデーション
- **枠**: レッド (border-red-500)
- **テキスト**: ライトレッド (text-red-300)

## アニメーション詳細

### カウントダウンタイマー
- **更新頻度**: 1秒ごと
- **トランジション**: 数字の切り替え時にスムーズ

### エントリーボタン
- **ホバー時**: スケール 1.05倍、シャドウ増強
- **クリック時**: 即座にローディング状態へ

### 参加者数プログレスバー
- **トランジション**: 500ms の ease-in-out
- **更新時**: スムーズに伸縮

### バトル結果モーダル
- **表示時**: 300ms のフェードイン + スケール (0.75 → 1.0)
- **報酬カウントアップ**: 1.5秒かけて0から実際の値まで
- **レベルアップ**: パルスアニメーション

### ステータスインジケーター
- **パルス**: 常時点滅（animate-pulse）
- **色**: グリーン (bg-green-400)

## レスポンシブデザイン

### モバイル (< 768px)
- カウントダウンタイマー: テキストサイズ縮小 (text-4xl → text-3xl)
- エントリーボタン: 幅いっぱい (w-full)
- モーダル: パディング縮小

### タブレット (768px - 1024px)
- 2カラムレイアウト維持
- 適度なパディング

### デスクトップ (> 1024px)
- 最大幅 4xl (max-w-4xl)
- 大きなテキストとボタン
- リッチなアニメーション効果
