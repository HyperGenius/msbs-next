## 実装の特徴

### デザイン
- ダークテーマ（グレー + 緑のアクセント）
- レトロなターミナル風UI
- アニメーション効果で視覚的フィードバック

### ユーザビリティ
- 直感的なミッション選択
- 明確な勝敗表示
- 履歴からの簡単な再確認
- レスポンシブデザイン（グリッドレイアウト）

### 技術的特徴
- SWR による効率的なデータフェッチング
- TypeScript による型安全性
- Next.js App Router 使用
- Clerk 認証統合
